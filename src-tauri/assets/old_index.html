Créer un exécutable embarqué

Gère plusieurs configurations (dossier & fichier de sortie), enregistre dans config.json. Le bouton lance /api/newServer et ouvre une nouvelle WebView dont le target = nom de la configuration sélectionnée.
Si vous laissez le port vide, le serveur sera arrêté automatiquement quand la WebView se ferme ou si la popup est bloquée.
